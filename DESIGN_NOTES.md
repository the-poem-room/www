# Design Notes

## Visual Identity

The site should feel:

- soft
- casually feminine
- floral
- ambient
- warm
- intimate
- literary
- emotionally thoughtful
- calm enough for long reading sessions

Avoid:

- startup aesthetics
- harsh black/white contrast
- loud colours
- sensory overload
- flashy gradients
- busy floral wallpaper
- overly formal archive styling

## Colour Direction

Use feminine colours more prominently while keeping the overall feeling soft.

Preferred colours:

- cream / ivory base
- blush pink
- rose
- dusty mauve
- soft lavender
- muted warm grey
- pale sage as an occasional balancing accent
- moonlit dark mode with rose, mauve, and lavender warmth

The current visual direction uses rose/lavender in:

- ambient floral background layers
- logo mark
- portrait image frame
- primary button
- soft cards and panels
- dark-mode glows

## Background

The background should be soft floral, ambient, and warm.

It should use:

- blurred bloom shapes
- petal-like forms
- blush and mauve washes
- subtle lavender glow
- low-contrast layering behind content

It should not become a dense pattern or distract from reading.

## Typography

Poems and literary headings:

- elegant serif font
- generous line height
- centred or comfortably constrained reading width

UI and navigation:

- clean sans-serif
- simple labels
- predictable navigation
- highly readable text

## Homepage Feel

The homepage should feel like an inviting entrance rather than a finished
archive.

Important areas:

- welcome message
- website purpose / functionality summary
- author bio
- portrait image area
- quiet path toward future poems

The tone should be personal, casual, and soft rather than overly polished or
corporate.

## Accessibility

Maintain:

- clear contrast
- readable text over backgrounds
- stable layouts
- responsive mobile design
- dark mode
- reduced-motion friendliness
- low sensory clutter

Softness should never make the site difficult to read.
