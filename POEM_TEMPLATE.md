# Poem Content Notes

This file is not a public-facing website template.

The current workflow is:

1. The user shares a poem directly with Codex.
2. Codex adds the poem to the website.
3. Codex adds any requested context, notes, tags, moods, or collection details.

## Optional Fields For Future Poem Pages

When adding a poem, these fields may be useful if the user provides them:

- Title
- Poem text
- Excerpt or short description
- Collection
- Categories
- Tags
- Mood
- Featured status
- Related poems

## Optional Marginalia Fields

Not every poem needs marginalia. Possible note fields:

- Writing Process
- Approximate Writing Time
- Written During
- Themes
- Influences
- Notes
- Form & Technique
- Personal Reflection

## Important Rule

Do not generate poems automatically.

Only display poems supplied directly by the user.
