# The Poem Room

A soft static homepage for a personal poetry archive, notebook, and reading
room.

## Local preview

Open `index.html` in a browser, or serve the folder with a small local server.

## Structure

- `index.html` - Homepage, welcome, purpose, bio, and poems introduction
- `styles.css` - Responsive light/dark visual system
- `script.js` - Dark mode preference handling

## Content note

The homepage is ready for a welcome message, a short bio, a portrait image,
and future poems supplied directly by the author.
