reading room

Group Chat (Muted)

**Logic:**
can we stay on topic

**Emotion:**
this *is* the topic

**Impulse:**
what if we just
changed our name
and moved somewhere
with low expectations

**Ethics:**
other people live there

**Impulse:**
ruined immediately

**Music:**
🎧 (voice note: 0:41)

**Logic:**
summary

**Music:**
no

**3am Thoughts:**
is anyone else scared that
we're wasting something
that doesn't renew

**Impulse:**
like time

**Emotion:**
or kindness

**Logic:**
or sleep

**Impulse:**
or a sandwich
we were emotionally invested in

**Ethics:**
can we not joke every time
something matters

**Impulse:**
i joke *because* it matters

**Emotion:**
i don't want to be efficient
i want to be understood

**Logic:**
those are not mutually exclusive

**Emotion:**
they are when someone's holding a clipboard

**Music:**
what if the answer isn't an answer
what if it's a tempo

**Logic:**
that's not helpful

**3am Thoughts:**
what if we're only overwhelmed
because we're paying attention

**Ethics:**
that doesn't make it wrong

**Impulse:**
it does in marking schemes

**System:**
*multiple inputs detected*

**Logic:**
please don't start

**Emotion:**
too late

**System:**
*Impulse has left the chat.*

**Emotion:**
that wasn't funny

**Music:**
statistically it was

**3am Thoughts:**
they'll be back

**System:**
*Impulse joined using a new account.*

**Impulse:**
hi sorry

**Ethics:**
you didn't apologise

**Impulse:**
i did. spiritually.

**Logic:**
that's worse

**Impulse:**
wow ok

**Emotion:**
can someone just say
it's okay to be like this

**System:**
*seen by everyone*

**System:**
*notifications silenced*

— Lilith
